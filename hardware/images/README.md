# Project Images

This directory contains photos and screenshots of the Office Control Panel project.

## Planned Images

- `panel_front.jpg` — Front view of assembled control panel
- `panel_side.jpg` — Side view showing enclosure depth
- `panel_boot_screen.jpg` — Boot screen with Southern IoT branding
- `panel_menu_lab.jpg` — Lab Room submenu on display
- `panel_menu_main.jpg` — Main Room submenu on display
- `wiring_internal.jpg` — Internal wiring view (ESP32 + TFT + encoder)
- `ha_dashboard.png` — Home Assistant dashboard showing controlled entities
- `nodered_flow.png` — Node-RED flow screenshot

## Photo Guidelines

- Minimum resolution: 1920x1080
- Format: JPEG or PNG
- Naming convention: `description_view.jpg`
- Remove any sensitive information (WiFi SSIDs, IP addresses on screen)
