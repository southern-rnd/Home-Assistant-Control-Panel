# 3D Enclosure Files

This directory contains 3D-printed enclosure design files for the Office Control Panel.

## Planned Files

- `panel_enclosure_top.step` — Top cover with display cutout and encoder hole
- `panel_enclosure_bottom.step` — Bottom base with USB-C port cutout and mounting holes
- `display_bezel.step` — Bezel ring around the ILI9341 display

## Design Notes

- Designed for PLA or PETG 3D printing
- Designed for M3 screws for assembly
- Encoder shaft hole: 6mm diameter
- Display cutout: 2.4" (approximately 50mm x 35mm active area)
- USB-C port cutout on bottom
- Ventilation slots on sides

## Manufacturing

- Recommended layer height: 0.2mm
- Infill: 20-30% for reduced weight
- Walls: 2 perimeters minimum
- Supports: none required (no overhangs > 45°)
